## 타입스트립트 객체

### 선언 방법

```js
// let 타입 선언부 = 객체
let[const, var] 변수명: {속성명: 타입, 속성명: 타입} = {속성명: 값, 속성명: 값}
```

## 인터페이스

> 타입을 정의한 규칙

- 객체를 여러번 재사용하거나 유지보수하기 편리

### 선언 방법

```js
// interface 인터페이스명 {속성명: 타입, ...}
interface 인터페이스명 {
  속성명: 타입;
  속성명: 타입;
  ...
}
```

### 사용 방법

```js
let 변수명: 인터페이스명 = {속성명: 값, ...}
```

## 인터페이스 상속(확장)

> 기존의 인터페이스 속성을 재사용해서 원하는 속성을 추가해 새로운 인터페이스를 생성

### 사용 방법

```js
interface 인터페이스명 extends 확장할_인터페이스명  {
  ...
}
```
